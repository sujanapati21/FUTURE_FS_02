# Mini CRM System (Full Stack Project)

A simple Customer Relationship Management (CRM) web application built using HTML, CSS, JavaScript, Node.js, Express.js, and MySQL.

---

## 📌 Project Overview

This project is a basic CRM system used to manage business leads. It allows users to add, view, and store lead information in a database with a secure login system.

---

## 🚀 Features

* Admin Login System
* Add New Leads
* View All Leads
* Store Data in MySQL Database
* Simple Secure Access (Frontend Authentication)
* REST API using Express.js

---

## 🛠️ Technologies Used

* HTML
* CSS
* JavaScript
* Node.js
* Express.js
* MySQL
